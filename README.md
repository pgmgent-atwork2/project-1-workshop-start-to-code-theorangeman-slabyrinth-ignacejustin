# De schatkist

Welkom in de schatkist! Hier vind je alle afbeeldingen die je nodig hebt om je labyrint te maken.

### Hoe krijg je deze op je computer?

1. **De makkelijkste manier:** Klik op het bestand **"assets.zip"** hierboven en druk daarna op de downloadknop (het pijltje naar beneden).
2. **De snelle manier:** Klik op de groene knop **[ <>Code ]** rechtsboven en kies **Download ZIP**.

### Wat moet je daarna doen?

Pak het ZIP-bestand uit en sleep de map met de naam "assets" naar je eigen project-map (die je "mijn_labyrint" hebt genoemd).
