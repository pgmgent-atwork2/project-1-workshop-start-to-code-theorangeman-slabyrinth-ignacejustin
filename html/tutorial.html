Tutorial: The Orange Man's Labyrinth

Introductie tot webontwikkeling

In deze tutorial bouw je stap voor stap een interactieve text adventure game. Je leert de basis van webontwikkeling met drie technologieën: HTML (structuur), CSS (opmaak) en JavaScript (logica). Aan het einde heb je een werkende game die je zelf kan uitbreiden.

Inhoud

  • Benodigdheden
  • Stap 1: HTML structuur
  • Stap 2: CSS styling
  • Stap 3: JavaScript logica

Benodigdheden

Wat hebben we allemaal nodig?

Voor deze tutorial heb je slechts twee tools nodig:

  • Browser: Chrome, Firefox of een andere moderne browser om je project te testen.
  • Visual Studio Code: een code-editor om je bestanden te schrijven.
    Klik hier om VS Code te downloaden

Maak daarna een projectmap aan met de naam mijn_labyrint. Dit wordt de basis van je project.

We hebben natuurlijk ook de afbeeldingen nodig voor onze game! Download de afbeeldingen en plaats ze in de map die je net hebt aangemaakt.
Klik hier om de afbeeldingen te downloaden.

Nu we alles hebben gedownload kunnen we eindelijk starten.

Open VS Code op je computer en klik op "File -> open folder". Kies daarna de folder (mijn_labyrint) die we net hebben aangemaakt.

Mappenstructuur

Maak nu onderstaande mappen en bestanden aan om je projectstructuur volledig op te zetten.
Om een map aan te maken klik je op "New folder" naast je map "mijn_labyrint". Om een bestand aan te maken klik je op "New file", of je kan dit doen via de rechtermuisknop.
Zorg ervoor dat de map assets op de juiste plaats staat in de mappenstructuur. Controleer goed of alle namen en locaties correct zijn, want zelfs een kleine fout kan ervoor zorgen dat je project niet werkt. Vermijd hoofdletters en spaties in de namen van mappen en bestanden, omdat dit fouten kan veroorzaken bij het inladen van je project.

Stap 3: JavaScript logica

JavaScript voegt interactie toe aan de website. In deze game zorgt het ervoor dat spelers keuzes kunnen maken en door het verhaal navigeren.

Wat doet deze code nu?

Selecteren van HTML-elementen voor tekst en keuzes

We bepalen eerst waar op de pagina de tekst en knoppen moeten verschijnen. De elementen #text en #options zijn in de HTML gedefinieerd met een id.

Beheer van de scènes in het spel

Hier wordt het verhaal opgeslagen. Elke scène bevat tekst en mogelijke keuzes die de speler naar een volgende stap brengen.

Functie voor het tonen van de scènes

Deze functie is de kern van het spel. Elke keer wanneer een speler een keuze maakt, wordt de juiste scène geladen en weergegeven.

Weergave van de tekst van de huidige scène

De juiste tekst van de actieve scène wordt opgehaald en weergegeven in het tekstgedeelte van de pagina.

Aanmaken van knoppen voor elke keuze

Voor elke beschikbare keuze wordt een knop aangemaakt zodat de speler een actie kan kiezen.

Plaatsen van knoppen in de HTML

De aangemaakte knoppen worden toegevoegd aan de juiste plaats in de webpagina.

Selecteer alle knoppen

Nadat de knoppen op de pagina staan, worden ze geselecteerd zodat er interactie aan toegevoegd kan worden.

Toevoegen van klikgedrag

Elke knop krijgt een klikfunctie. Wanneer een speler klikt, wordt bepaald naar welke scène hij wordt gestuurd.

Start van het spel

Deze regel zorgt ervoor dat het spel start bij de eerste scène zodra de pagina wordt geladen.

function startGame(sceneKey) {
  const scene = gameData[sceneKey];

  $text.innerHTML = scene.text;
  let html = "";

  scene.options.forEach((option) => {
    html += `<button data-next="${option.next}">${option.text}</button>`;
  });

  $options.innerHTML = html;

  const buttons = document.querySelectorAll("button");
  buttons.forEach((button) => {
    button.addEventListener("click", () => {
      startGame(button.dataset.next);
    });
  });
}

startGame("entrance");
